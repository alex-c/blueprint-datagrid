Blueprint Datagrid

An enhanced table component for Blueprint 4.

PlaygroundGetting Started

Common Table Features

Blueprint Datagrid wraps the basic HTMLTable and adds common table features like sorting, filtering and pagination.

Faithful in Style

Built with Blueprint components and styling, Blueprint Datagrid stays faithful to the look-and-feel of the library in both dark and light mode.

Easy to Use

Blueprint Datagrid aims to be easy to use, but still allows customization.
